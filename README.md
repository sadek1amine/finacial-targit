## 📖 About

**Financial Target** is a modern expense and income tracking application that helps users manage their personal finances with a clean, intuitive, and beautiful interface.

The application combines powerful budgeting tools with a unique **winter-inspired design**, featuring cool colors, snow-themed visuals, and a friendly **polar bear mascot** that represents financial discipline, stability, and resilience.

Whether you're tracking daily expenses, planning monthly budgets, or monitoring your savings goals, Financial Target makes personal finance simple and enjoyable.

## 🚀 Getting Started

### Clone the project

```bash
git clone https://github.com/yourusername/financial-target.git
```

### Install dependencies

```bash
npm install
```

### Start development server

```bash
npm run dev
```

### Build for production

```bash
npm run build
```
